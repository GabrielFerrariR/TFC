## Sobre

Analise o placar geral do campeonato, acompanhe o resultado das partidas e torça pela classificação do seu time nessa aplicação.


Este projeto envolve uma aplicação front-end disponibilizada pela [Trybe](www.betrybe.com), alimentada por uma API desenvolvida por mim.

### Tecnologias utilizadas

- HTML5;
- CSS3;
- React com componentes funcionais;
- Context API para gerenciamento de estado;

### O que é possível fazer

- Logar com um usuário cadastrado;
- Visualizar a tabela de classificação atualizada;
- Modificar o placar de partidas em andamento;
- Finalizar partidas em andamento.
